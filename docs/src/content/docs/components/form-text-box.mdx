## Import

```js
import { FormTextBox } from "basecamp/components";
```

## Usage

```mdx title="src/pages/index.astro"
import { FormTextBox } from "basecamp/components";

      <FormTextBox
        id="email"
        type="email"
        label="Email"
        placeholder="name@example.com"
        required={true}
      />
```

The above code generates the following on the page:

      <FormTextBox
        id="email"
        type="email"
        label="Email"
        placeholder="name@example.com"
        required={true}
      />
